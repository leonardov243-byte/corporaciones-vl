# 🌐 Proyecto de Automatización Cloud (Python + AWS + Git)

Este proyecto demuestra la conexión y el despliegue de código desde un entorno local hacia una instancia en la nube.

### 🎯 Objetivo
Aprender a gestionar el flujo de archivos entre tres puntos clave: mi computadora personal, un repositorio remoto en GitHub y un servidor **Amazon EC2** ubicado en EE.UU.

### ⚙️ ¿Cómo funciona?
1. **Lógica en Python:** Creé un script que genera registros automáticos (logs) para monitorear la actividad del sistema.
2. **Control de Versiones:** Uso Git para "subir" los cambios desde VS Code hacia internet.
3. **Despliegue Real:** Me conecto por terminal al servidor de AWS y "jalo" el código para ejecutarlo en un entorno de producción real.

### 🛠️ Herramientas utilizadas
* **Python 3** (Lógica de programación)
* **Git/GitHub** (Transporte de código)
* **AWS EC2** (Servidor en la nube - Linux)

---
*Este es un proyecto de práctica para dominar la arquitectura de sistemas distribuidos.*
